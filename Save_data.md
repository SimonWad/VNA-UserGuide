# Saving data files on external USB

The main apeal of the VNA is capturing data for your projects. So this guide wil focus on saving to an external USB as this is the easiest way of getting data from device to device.

The Ascociated USB contains a **user guide** that is handy for exploring the features of the FieldFox.

> **Notice:** the guide is for N99xxA, meaning that not everything mentioned in the PDF is applicable to our device, the **FieldFox N9951A**.

If anything is unclear please refer to the user guide or the offical documentation.

## CSV data to USB

> **! Do not use two storage USB devices simultaniusly!** If you wish to transfer files from one USB to another, please do so on your own PC.

CSV (**C**omma-**S**eperated **V**alues) is an easy way of saving and reviewing data, so for the most part this is the recomended file type to use.

When you are ready to save your data file:

&rarr; Press **9: Save/Recall**

&rarr; Press **File Type** on the screen select buttons

&rarr; Press **More** on the screen select buttons

&rarr; Select **Data (CSV)**

&rarr; Press **Device** on the screen select buttons

&rarr; Select **USB**

&rarr; Press **Save** on the screen select buttons

&rarr; Name your file appropriately

> Example could be `50Hz_Notch_Filter_USER.CSV` for the person USER@sdu.dk

&rarr; Press **Done** and the file is saved

&rarr; The USB can now be removed and your file can be transfered

### Copying or moving files

It is possible to copy a file from `[INTERNAL]:` to USB, in the event that your file was saved to the wrong folder.

> **NOTE:** Do not move any files that is not yours, if any accidental operations happen. Write down what was Erased/Moved and contact the equipment manager or your supervisor.

&rarr; Press **9: Save/Recall**

&rarr; Press **More** on the screen select buttons

&rarr; Press **Manage Files** on the screen select buttons

If you do not see your datafile, make sure you are in the right directory.

&rarr; Highlight your desired file with the scroll disc

&rarr; Press **Copy** on the screen select buttons

&rarr; Select **Copy To USB**

&rarr; Select the `[USB]:\USBDisk\DATA_FILES_USB` directory

&rarr; Press **Do Copy** on the screen select buttons and the file should be moved to the USB device

> Before you delete the file from `[INTERNAL]:` make sure that the file on the USB is correct and working. Save yourself the headache.
